# Salut, moi c'est Freddy Choudja

Je suis **Freddy Choudja**, développeur passionné par la création de solutions
web, l'automatisation et les outils numériques utiles au quotidien.

J'aime construire des projets simples, propres et pratiques, avec une attention
particulière pour la qualité du code, l'apprentissage continu et le partage de
connaissances.

## Ce que je fais

- Développement d'applications web
- Automatisation de tâches répétitives
- Expérimentation avec des outils modernes
- Amélioration continue de mes compétences techniques
- Contribution à des projets personnels et open source

## Domaines qui m'intéressent

- Développement full-stack
- JavaScript et technologies web
- Python
- Bases de données
- Git, GitHub et GitHub Actions
- Déploiement et maintenance de projets
